# Receptnik

## 🍳 Moderná kuchárska kniha

Receptnik je webová aplikácia na správu receptov s podporou AI návrhov.

### 🚀 Inštalácia

```bash
git clone https://github.com/tvoje-meno/receptnik.git
cd receptnik
pip install -r requirements.txt
```

### 🔧 Spustenie

```bash
export DEEPSEEK_API_KEY=tvoj_kluc
python app.py
```
